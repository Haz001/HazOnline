How to install Pygames and python

  1. Install Python

    Install Python 3.X from Python.org.

    To install it:
    1. Open the installer
    2. Check 'Add Python 3.X to PATH'
    3. Click 'Install Now'
  2. Install Pygame

    To install Pygame you will be using pip.

    1. Open cmd.exe
    2. Type:
      1. pip install pygame
  3. Done

    Your done, now go play or make Pygame games.
